# 🚀 Job Portal (Django Full-Stack Project)

A full-stack job portal web application built using Python (Django), HTML, CSS, and JavaScript.
---

## 🔥 Features

- 🔐 User Authentication (Register / Login / Logout)
- 📄 Job Listing Page
- 🔍 Job Search (JavaScript-based filtering)
- 📌 Job Detail Page
- 📨 Apply for Jobs with Resume Upload
- 📊 User Dashboard (Track applied jobs)
- 🧑‍💼 Admin Panel for managing jobs and applications

---

## 🛠️ Tech Stack

- Backend: Python, Django
- Frontend: HTML, CSS (Bootstrap)
- JavaScript: Search & UI interaction
- Database: SQLite

---

## 📸 Screens Included

- Job List Page
- Job Detail Page
- Apply Form
- Dashboard
- Admin Panel
